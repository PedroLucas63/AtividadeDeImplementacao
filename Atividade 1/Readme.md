# Calculadora Básica
O seguinte projeto de código visa produzir uma calculadora que execute as operações de soma, subtração, multiplicação e divisão na linguagem de programação JavaScript.

## Funções:
- [ ] 🔢 Receber dois números.
- [ ] ➕ Somar.
- [ ] ➖ Subtrair.
- [ ] ✖️ Multiplicar.
- [ ] ➗ Dividir.
- [ ] 🟰 Exibir o resultado.

## Tecnologias:
- HTML
- JavaScript
- Tailwind CSS
